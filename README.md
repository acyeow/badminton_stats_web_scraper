# Python Scraper for Scraping Badminton Stats

## Quickstart

### Create a venv

```
python3 -m venv venv python=3.10.12
```

### Activate the venv
```
source venv/bin/activate
```

### Install requirements

```
pip install -r requirements.txt
```

### Run the script

```
python3 scraper/scraper.py
```
